# Recipes Web App

Готовое веб-приложение по предметной области «Рецепты».

## Возможности
- Публичный просмотр рецептов без регистрации.
- Фильтр по категориям (12 категорий) и поиск.
- Регистрация/вход/выход по логину и паролю.
- Восстановление пароля по коду из Gmail.
- Личный кабинет автора.
- Создание/редактирование/удаление рецептов.
- Загрузка обложки и картинок для шагов.
- Адаптивный интерфейс: на ПК контейнер по центру как мобильное приложение.

## Запуск
1. Установите Node.js 18+.
2. Установите зависимости:
   npm install
3. Задайте переменные окружения в системе (через systemd/Nginx panel/хостинг-панель):
   - `PORT`
   - `SESSION_SECRET`
   - `GMAIL_USER`
   - `GMAIL_APP_PASSWORD`
4. Запустите сервер:
   npm start
5. Откройте: `http://localhost:3000`

## Почта Gmail
Для отправки кодов восстановления используется Gmail SMTP.
Нужен именно `GMAIL_APP_PASSWORD` (пароль приложения), а не обычный пароль от аккаунта.
